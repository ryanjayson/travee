- A trip can be publish (future)
- Can invite multiple members in a trip
- Trip creator is the owner
- An owner can set roles to members (owner, planner, contributor, member)
- Planners and contributors can modify a trip 
- Members can only view a trip
- A member can extend or update trip sections or activities, but only visible to them
- A member can have its own itinerary that is visible only to them
- A member can have its own checklist that is visible only to them
- A member can have its own budget that is visible only to them
- A member can have its own notes that is visible only to them
- A member can add expense to a section or activity
